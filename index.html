GestiZone

Vue · Laravel · MySQL

GestiZone is a full-stack incident and task management system designed for internal company use. The application includes user authentication, role-based access control, and a complete workflow for creating, assigning, updating, and closing incidents.

I developed the entire project from scratch, including database design, backend API, frontend interface, and application logic. The backend is built with Laravel following REST principles, while the frontend uses Vue to provide a responsive and dynamic user experience.

The system supports different user roles (standard users and administrators), real-time data updates, file attachments, and structured status management. This project reflects how I approach real-world problems: clean structure, scalability in mind, and practical solutions.
