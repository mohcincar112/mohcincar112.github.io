Projects

GestiZone

Vue · Laravel · MySQL


GestiZone is a full-stack incident and task management system built for internal company operations. It allows users to create, assign, and track incidents in a structured workflow, supporting multiple user roles, including standard users and administrators.


I built GestiZone from the ground up, designing the database, implementing backend logic with Laravel, and creating a dynamic frontend interface with Vue. The project follows RESTful principles and includes authentication, role-based access control, real-time updates, and file attachments.


The system demonstrates my approach to software development: designing scalable architecture, writing maintainable code, and building features that address real-world needs. It also shows my ability to handle both frontend and backend responsibilities, and integrate them into a cohesive full-stack solution.
